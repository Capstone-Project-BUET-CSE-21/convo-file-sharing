# Convo Confidentiality in File Sharing

Backend microservice for confidential file sharing, built with **Spring Boot (Java)**.

**Stable Branch:** `main`

- Frontend: https://github.com/Capstone-Project-BUET-CSE-21/convo-frontend
- Main Backend: https://github.com/Capstone-Project-BUET-CSE-21/convo-backend
- Audio Watermarking: https://github.com/Capstone-Project-BUET-CSE-21/convo-audio-watermarking

## Running Locally
```
cd file-sharing
./mvnw spring-boot:run
```
Server starts at `http://localhost:8082`
